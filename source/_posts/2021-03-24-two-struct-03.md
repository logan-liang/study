Go语言中您只有for结构可以使用。不要小看它，这个for结构比其它语言中的更为灵活。

### 基于计数器的迭代

```
    for 初始化语句；条件语句；修饰语句{}
```

示例1

```
    for i:=0;i<5;i++{
        //do something
    }
```

示例2

```
    for i,j:=0,N;i<j;i,j=i+1,j-1{}
```
